$plugins.uiSlider


$plugins.uiSlider({ 
    id:"uiSlider", vertical:false, reverse:false, range:false, acc:true, callback:sliderCallback, stepname: false
    now:[500], step:10, min:0, max:1000, 
    tooltip:true, unit:'만원', txt_s:'이하', txt_e:'이상' 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-sider의 아이디명 기입
vertical: [boolean] false 가로/세로 모드 설정
  • true: 세로모드
  • false: 가로모드
reverse: [boolean] false 순서 설정
  • true: 역순으로 설정
  • false: 기본순
range: [boolean] false 순서 설정
  • true: 범위설정 (최소,최대)
  • false: 기본
acc: [boolean] false 순서 설정
  • true: 접근성을 위해 셀렉트 박스와 연결
  • false: 미사용
now: [array] [0] 현재 설정 값, range가 true일때 [최소, 최대] 2개의 값
step: [number] 10 단계 설정
min: [number] 0 최소 값
max: [number] 100 최대 값
stepname: [boolean][array] false 구분부분 텍스트지정
  • ['step1', 'step2', ......]
  • false: 미사용
tooltip: [boolean] false 툴팁 생성 여부 설정
  • true: 툴팁생성
  • false: 미사용
unit: [string] '' 툴팁에 단위명 추가 예)0000만원
txt_s: [string] '' 단계에 첫번째 오브젝트에 추가 예) 100 이하
txt_e: [string] '' 단계에 첫번째 오브젝트에 추가 예) 1000 이상
callback: [function] function 콜백값으로 아이디와 최소/최대 값, 최소/최대버튼 위치값 %
