제 1 조 (목적)

본 약관은 SK 텔링크 주식회사(이하 "회사"라 합니다)에서 제공하는 7mobile 관련 서비스 이용을 위한 회원과 회사간의 권리, 의무 및 책임사항 등 기본적인 사항의 규정을 목적으로 합니다.

제 2 조 (용어의 정의)

  • ① 본 약관에서 사용하는 용어의 정의는 다음과 같습니다.
    1. 1. 서비스 : sk7mobile.com 및 모바일 7mobile app. 이외에 sk7mobile.com 과 직접 연동되어 온라인에서 이용할 수 있는 서비스
    2. 2. 회원 : 사이트에 접속하여 본 약관에 동의하고, ID 정책에 따른 ID(고유번호)와 Password(비밀번호)를 발급 받은 고객. 회원의 자격 및 권한 등은 본 약관에서 정한 바에 따라 일부 제한될 수 있습니다

			NETIVE.uiLayPopup.open({ id:'exeLaypop_1', link: 'ajax로 불러드릴 파일 주소' });
			NETIVE.uiLayPopup.close({ id:'exeLaypop_1', endfocus: '아이디' });
				[id] 아이디명 
				 - Type: String
				[link] ajax로 파일을 불러올 주소
				 - Type: String
				[endfocus] 아이디명. 값 설정 시 해당 위치로 포커스 이동.
				 - Type: String
				 - Default: document.activeElement / '아이디명'

				[id] 해당 레이어 팝업 아이디명 
				 - Type: String
				[name] 쿠기 이름 
				 - Type: String
				[day] 유효기간 일자 
				 - Type: Number
