Naming

class 네이밍 일반 규칙

이름은 영문 소문자, 숫자, 하이픈(-)으로 작성한다.

이름의 시작문자는 영문 소문자로만 시작한다.


.btn-base
.list2

class 네이밍 조합 규칙

  • 하이픈(-)은 2개 이상의 단어 조합할 때만 사용한다.
  • 하이픈(-)조합은 최대 2단계를 초과하지 조합한다.
  • 단어와 숫자 조합은 하이픈(-)을 생략한다.
  • 숫자는 1부터 시작하며 '01' 과 같이 앞에 '0'을 붙여 사용하지 않는다.

.btn-base-lg
.btn-base-med
.btn-base-sm
.box-dark
.n1

class 네이밍 자바스크립트 전용

ui- 로 시작하는 클래스는자바스크립트에서만 사용한다.


.ui-name

ID 네이밍 규칙

이름은 camel casing(카멜 표기법)으로 작성한다.

이름의 시작문자는 영문 소문자로만 시작한다.

ID에는 어떠한 스타일도 주지 않는다.


#btn
#btnClose1

class 예약어

기본적으로 주로 사용하는 class에 대해 구분지어 예약어를 사용한다.

class 예약어
예약 접두어 설명
.base- 전체적으로 기본적으로 사용하는 스타일
  • .base-skip : 공통 skip 메뉴
  • .base-modal : 공통 modal layer popup 공간
  • .base-wrapper : skip, modal 제외한 영역을 감싸는 공간
  • .base-header : header 영역
  • .base-main : main 영역(모든 문서에 반복적으로 있는 어떠한 컨텐츠라도 제외되어야 하며 문서 안에 가장 유일한 요소여야 합니다.)
  • .base-side : side 영역
  • .base-content : 주요 내용이 들어가는 영역
  • .base-footer : footer 영역
.btn- button에 관련된 기본 스타일
.tbl- table에 관련된 스타일
.box- box에 관련된 스타일
.inp- input에 관련된 스타일
.list- list에 관련된 스타일
.page- 해당 페이지에만 관련된 스타일
