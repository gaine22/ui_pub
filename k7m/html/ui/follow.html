■■javascript code
NETIVE.uiFollow.inner({ id: 'exeFix2', ps: 'bottom', fix: false, add: 'baseFooter' });
	[id] 아이디명 
	 - Type: String
	[ps] 고정될 위치 
	 - Type: String
	 - Default: bottom / top, bottom
	[fix] 처음부터 고정일지 설정. false 이면 relative > fixed로 변경, true 이면 fixed > relative로 변경
	 - Type: Boolean
	 - Default: true / true, false
	[add] 아이디명. ps값이 'top'일 경우 해당 아이디의 높이값을 구해 fixed 될 위치 설정, ps값이 'bottom'일 경우 해당 아이디의 위치값을 구해 한계점 설정
	 - Type: String
	 - Default: false / false, 아이디명 
	[callback] 함수명. function callback(v) { v.scrolltop //현재 스크롤 값 받아옴. }
	 - Type: String
