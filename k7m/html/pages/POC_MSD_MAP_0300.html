휴대폰결제 내역안내
  1. 01. 휴대폰결제 내역안내
    • - 소액결제 내역은 최근 1년까지 조회 가능합니다.
  2. 02. 결제구분
    • - 자동결제 : 한 번의 인증으로 다음달부터 추가 인증 없이 자동으로 이루어진 결제입니다.
    • - 묶음결제 : 온라인 쇼핑몰 등의 장바구니 형태로 여러 상품을 묶어서 한번에 진행된 결제입니다.
  3. 03. 자동결제 차단/해제
    • - 자동결제 차단/해제는 신청 월 말일의 설정 상태를 기준으로 신청 월 다음 달부터 적용됩니다. 고객님이 7월 25일에 "자동결제 차단하기"를 설정했을 경우,
      7월 31일 기준으로 "자동결제 차단" 상태이므로 8월부터 자동결제 차단이 적용됩니다.
    • - 자동결제 차단/해제는 월 단위로 적용됩니다. 고객님이 7월 25일에 "자동결제 차단하기"를 설정하였다가 8월 1일에 "자동결제 차단해제"를 설정할 경우,
      8월분은 자동결제 차단이 되고 9월부터 자동결제 차단해제가 됩니다.
    • - 자동결제 차단/해제는 사용처(콘텐츠 제공사) 단위로 적용되므로, 동일 사용처의 특정 콘텐츠에 대한 자동결제 차단/해제를 희망하실 경우에는
      사용처(콘텐츠 제공사)의 고객센터로 문의하시기 바랍니다.
    • - 자동결제 차단/해제는 사용처(콘텐츠 제공사)에서 과금 시도 시, 고객이 자동결제 차단신청을 한 상태이면 SK7Mobile에서 임시로 과금되지 않도록
      차단하는 서비스입니다. 자동결제 차단하기는 자동결제 내역이 있을 경우 신청이 가능합니다. 자동결제를 완전히 차단하시고자 하는 고객님께서는
      사용처(콘텐츠 제공사) 고객센터를 통해 자동결제를 해지하셔야 합니다.
  4. 04. 공급가액 및 부가세 표기
    • - 2009.05.01 부터 결제된 금액에 대해서 분리하여 표기(일부 결제대행사의 경우 적용시기 지연가능)하며, 2009.04.30 포함 이전의 결제건은 공급가액란에
      부가세를 포함한 금액을 표기합니다.
