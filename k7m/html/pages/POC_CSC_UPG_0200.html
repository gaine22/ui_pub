명의도용방지서비스

명의도용 방지 서비스

각종 통신서비스에 가입하거나 명의변경을 통해 양도받을 경우 그 사실을 본인 명의의 SMS로 알려주고,
홈페이지를 통해 실시간 개통현황을 확인할 수 있는 서비스 입니다.

M-safer 홈페이지 바로가기

M-safer 서비스 구상도

  1. 대리점, 인터넷을 통해 신규가입 (유선전화, 이동전화, 무선인터넷, 초고속 인터넷, 인터넷 전화, 방송서비스)
  2. 명의도용방지 서비스로 개통사실 통보
  3. 개통사실안내 2가지 방법, 방법1 - e-mail를 통한 고객에게 개통사실 안내, 방법2 - 이동통신사업자에게 개통사신 안내 sms 발송요청 후, 이동통신사가 고객에게 개통안내 sms

스미싱피해 예방안내

스마트폰을 통한 스미싱 범죄를 사전에 방지할 수 있도록
스미싱 피해 예방 방법을 알려드립니다.

스미싱(Smishing) 이란

문자메시지(SMS)와 피싱(Phishing)의 합성어

  • '무료쿠폰', '돌잔치 초대장',
    '모바일 청첩장' 등을 내용으로 하는
    문자메시지 내 인터넷 주소 클릭
  • 악성코드가
    스마트폰에 설치
  • 피해자가 모르는 사이에
    소액결제 피해 발생
    또는 개인·금융정보 탈취

피해유형

스미싱에 이용된 변종 악성코드는 소액결제 인증번호를 가로채는 것에 그치지 않음 최근에는 피해자 스마트폰에 저장된 주소록 연락처,
사진(주민등록증·보안카드 사본), 공인인증서, 개인정보 등까지 탈취하므로 더 큰 금융범죄로 이어질 수 있습니다.

피해 예방 방법

  1. 출처가 확인되지 않은 문자메시지의 인터넷주소를 클릭 금지
    • 지인에게서 온 문자메시지라도 인터넷주소가 포함된 경우 클릭 전에 전화 확인
  2. 미확인 앱이 함부로 설치되지 않도록 스마트폰의 보안설정 강화
    • 스마트폰의 보안설정 강화방법 : 환경설정 > 보안 > 디바이스 관리 > ‘알 수 없는 출처’에 V체크가 되어 있다면 해제
  3. 이동사 고객센터에 전화하거나 이통사 인터넷 홈페이지를 이용하여 소액결제를 원천적으로 차단하거나 결제금액 제한
    • 자신의 스마트폰으로 114를 눌러 상담원과 연결해도 소액결제 차단 가능
  4. 스마트폰용 백신프로그램을 설치하고 주기적으로 업데이트
  5. T스토어 · 올레마켓 · LGU+ 앱스토어 등 공인된 오픈마켓을 통해 앱 설치
  6. 보안강화 · 업데이트 명목으로 금융정보를 요구하는 경우 절대 입력 금지

대처요령

피해구제
  • 금융기관 콜센터 전화
    • 지인에게서 온 문자메시지라도 인터넷주소가 포함된 경우 클릭 전에 전화 확인
  • 악성파일 삭제
    • 스마트폰의 보안설정 강화방법 : 환경설정 > 보안 > 디바이스 관리 > ‘알 수 없는 출처’에 V체크가 되어 있다면 해제

      - 삭제되지 않는 경우, ⅰ) 휴대전화 서비스센터 방문 또는 ⅱ) 스마트폰 초기화

  • 악성코드 삭제
    • 자신의 스마트폰으로 114를 눌러 상담원과 연결해도 소액결제 차단 가능

      - 한국인터넷진흥원 ‘보호나라(www.boho.or.kr)' 서비스를 통해 ‘PC원격 점검’ 이용

금전 피해가 발생하지 않는 경우
  • 스마트폰에 설치된 악성파일 삭제
  • 해당 이통사 제공 예방서비스 이용

피싱피해 예방안내

피싱 피해예방 요령과 피해 시 대처 방법에 대해 알려드립니다.

피싱(Phishing) 이란?

개인정보(Private data)와 낚시(Fishing)의 합성어

  1. 금융기관을 가장한
    이메일발송
  2. 이메일에서 안내하는
    인터넷주소 클릭
  3. 가짜 은행사이트로
    접속유도
  4. 보안카드번호
    전부 입력 요구
  5. 금융정보 탈취
  6. 범행계좌로 이체

피해유형

“9월 카드 거래내역”이라는 제목의 이메일에서 안내하는 인터넷주소를 클릭하자 가짜 은행사이트로 접속되면서 보안카드번호 전부를 입력한 결과,
범행계좌로 무단 이체

피해 예방 방법

  1. OTP(일회성 비밀번호생성기), 보안토큰(비밀정보 복사방지) 사용
  2. ‘출처불명’ 또는 금융기관 주소와 ‘다른 주소’로 발송된 이메일 즉시 삭제
    • 이메일 첨부파일에 확장자가 ‘.exe, .bat, .scr 등 압축파일’이면 열람 금지
  3. 보안카드번호 전부를 절대 입력하지 말 것
  4. 사이트 주소의 정상 여부 확인
  5. 보안강화 · 업데이트 명목으로 금융정보를 요구하는 경우 절대 입력 금지
    • 가짜 사이트는 정상 사이트 주소와 유사하나, 문자열 순서·특수문자 삽입 등에서 차이 있음

대처요령

금전피해가 발생한 경우
  • 피해구제
    • 신속히 112센터나 금융기관 콜센터를 통해 지급정지 요청 후 해당 은행에 경찰이 발급한 ‘사건사고 사실확인원’을 제출하여 피해금 환급 신청
  • 피해금 환급 과정(전기통신금융사기 피해금 환급에 관한 특별법)
    1. 지급정지신청
      • 피해자가 112 또는 금융기관 콜센터에 전화, 지급정지 신청
      • 해당 은행에서 지급정지 조치

        * 경찰에서 발급한 사건 사고 사실 확인원 제출

    2. 명의자 채권소멸

      [주관 : 금융감독원]

      • 명의자채권소멸 개시 공고
      • 공고 2개월 후 채권소명

        * 2개월 경과 전까지 명의인 이의 제기 가능

    3. 피해자 환급

      [주관 : 금융감독원]

      • 금융감독원은 명의자 채권 소멸 후 14일 내 환금금액 결정하여 금융기관 피해자 통지
      • 금융기관은 피해금 지급

    - 가짜사이트 신고 : 한국인터넷진흥원 보호나라(www.boho.or.kr)에 신고

금전 피해가 발생하지 않은 경우
  • 수신한 이메일 삭제
  • 입력했던 금융정보들은 해당 금융기관을 통해 변경
  • OTP(일회성 비밀번호생성기) 사용

파밍피해예방안내

파밍피해 예방안내

피싱의 진화된 형태 파밍사이트 식별법과 대처방법에 대해 알려드립니다.

파밍(Pharming) 이란?

악성코드에 감염된 사용자PC를 조작하여 금융정보를 빼내는 것

  • 사용자PC가
    악성코드에 감염됨
  • 정상 홈페이지에 접속해도
    피싱(가짜)사이트로 유도
  • 금융정보 탈취
  • 범행계좌로 이체

- (피싱사이트) ‘http://*Kb*bank.com’ 등 정상 홈페이지로 가장하여 금융정보(보안카드번호 전부) 입력을 요구, 신종금융사기의 주요 범행수단

피해유형

피해자 184명이 동일한 파밍 수법으로 금융정보가 탈취되고 공인인증서가 재발급되어 불시에 13억원이 무단 이체됨

피해 예방 방법

  1. OTP(일회성 비밀번호생성기), 보안토큰(비밀정보 복사방지) 사용
  2. 컴퓨터 · 이메일 등에 공인인증서, 보안카드 사진, 비밀번호 저장 금지
  3. 보안카드번호 전부를 절대 입력하지 말 것
  4. 사이트 주소의 정상 여부 확인

    가짜 사이트는 정상 사이트 주소와 유사하나, 문자열 순서 · 특수문자 삽입 등에서 차이 있음

  5. 윈도우, 백신프로그램을 최신 상태로 업데이트하고 실시간 감시상태 유지
  6. 전자금융사기 예방서비스(공인인증서 PC지정 등) 적극 가입(9. 26 전면 시행 예정)
  7. ‘출처불명’ 한 파일이나 이메일은 즉시 삭제 및 무료 다운로드 사이트 이용 자제

스팸메시지대응하기

스팸메시지 대응하기

휴대폰, 전자우편, 메신저 등 다양한 매체를 통해 수신되는
불법스팸 메시지들에 대한 대처법을 알려 드립니다.

스팸 메시지(Spam Message)란?

수신자의 동의 없거나 수신동의 의사를 철회하였음에도 불구하고 불특정다수에 대량으로 발송되는 문자메시지 도는 음성전화로써,
광고/대출 등 무차별적인 정보 제공을 목적으로 행해지는 행위를 뜻 함

스팸 메시지 차단법

스팸등록을 통한 스팸 메시지 차단
  • 스마트폰이 가진 기본 기능 사용 (안드로이드 기기만 해당)
    스팸 메시지를 클릭하면 메시지 옵션 버튼 창이 나타나며,
    버튼 창에서 스팸 등록 버튼을 누르게 되면 문자를 보낸 번호가
    스팸으로 등록되어 차단됨
  • 하지만 다른 번호로 온 스팸 문자는 차단되지 않는 단점이 있으며,
    스팸 메시지 번호를 하나하나 등록해야 하는 번거로움 있음
어플리케이션을 통한 스팸 메시지 차단
스마트폰에 설치할 수 있는 어플리케이션을 사용
  • ‘후후- 대한민국 스팸 잡는 1등 전화’ (안드로이드 기기 대상), ‘후후- 모르는 전화번호, 스팸번호 확인’ (아이폰 기기 대상) 어플리케이션 설치
  • 어플리케이션 특징
    • 발신자 전화번호 정보 즉시 확인
      • 대한민국 2천만 개 이상의 전화번호 정보 즉시 확인
      • 상호, 업종, 지역 등 전화번호의 상세정보 확인
      • 공중전화, 발신전용 전화 등 전화번호 종류 확인
    • 스팸전화 등 악성 전화번호 정보 확인
      • 스팸 전화, 스팸 메시지, 보이스피싱, 스미싱 번호 등 확인
      • 후후 이용자들 간 실시간 스팸정보 공유
      • 실시간 스팸전화 탐지 서비스, 해외 전화번호 정보 확인, 실시간 스미싱 탐지 기능, 전화 메시지 전송 등 기능 보유
KISA 불법스팸대응센터 신고를 통한 스팸 메시지 차단
  • KISA 불법스팸대응센터(https://spam.kisa.or.kr/integration/main.do) 에서는 국민 대상으로 스팸 차단 방법 등을 안내하고 ‘정보통신망 이용 촉진 및 정보 보호 등에 관한 법률’에서 정하고 있는 불법스팸의 신고를 연중 접수하여 처리
  • 스팸 피해를 사전에 예방하기 위한 각종 인식 제고 활동
