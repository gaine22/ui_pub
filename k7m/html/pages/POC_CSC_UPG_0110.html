불법TM 신고 안내

SK텔링크 7mobile은 불법 TM 영업근절을 통하여 고객님의 권익을 보호하고자
알뜰폰 통신사(MVNO)의 불법 TM 영업행위에 대한 신고를 진행하고 있습니다.
단, 불법 TM 신고 건이 아닌 경우 (녹취 증빙 및 단말일련번호가 확인되지 않는 경우)는 가입하신 통신사 고객센터(114)로 문의하시길 바랍니다.

이동통신서비스 불법TM(Telemarketing:전화 권유 판매) 이란?

알뜰폰 통신사(MVNO) 불법TM 유형
  • ARS 등 녹음된 음성의 기술적 조치를 통한 TM
  • 내용이 허위/과장 광고인 경우 (SKT 우수고객행사, 단말무료 제공, 최저가 요금제 등)
  • 광고성 정보 전송자의 명칭/연락처를 밝히지 않는 행위 (발신전용 전화사용, TM업체 소속 미확인)

불법TM 신고 절차

신고 대상
  • 알뜰폰 사업자의 이동전화 불법 TM 영업 건

    단, 각 통신사 별 공식 기변센터가 자사의 가입자를 대상으로 기변 안내하는 영업행위는 제외

  • 가입 조건을 허위로 안내하거나, 타 통신사의 위약금 및 잔여할부금 대납 등을 안내하는 경우
신고절차
신고절차 (이미지 위치)

  • ※ SMS 알림으로 고객님의 답변완료를 알려드립니다.

  • ※ 상담내역 조회 시 필요한 비밀번호 영문, 숫자 조합 10~20자리로 입력해주세요.

  • 신고 접수 시 녹취 파일 및 휴대폰 일련번호 사진촬용 자료는 개인정보 보호를 위하여 이메일(mvnovoc@sk.com)로 송부하여 주시길 바랍니다.

스미싱 피해 예방 방법을 알려드립니다.

스미싱(Smishing) 이란

문자메시지(SMS)와 피싱(Phishing)의 합성어

스미싱 (이미지 위치)

피해유형

스미싱에 이용된 변종 악성코드는 소액결제 인증번호를 가로채는 것에 그치지 않음 최근에는 피해자 스마트폰에 저장된 주소록 연락처,
사진(주민등록증·보안카드 사본), 공인인증서, 개인정보 등까지 탈취하므로 더 큰 금융범죄로 이어질 수 있습니다.

피해 예방 방법 (@디자인필요)

  1. 1. 출처가 확인되지 않은 문자메시지의 인터넷주소를 클릭 금지
    • - 지인에게서 온 문자메시지라도 인터넷주소가 포함된 경우 클릭 전에 전화 확인
  2. 2. 미확인 앱이 함부로 설치되지 않도록 스마트폰의 보안설정 강화
    • - 스마트폰의 보안설정 강화방법 : 환경설정 > 보안 > 디바이스 관리 > ‘알 수 없는 출처’에 V체크가 되어 있다면 해제
  3. 3. 이동사 고객센터에 전화하거나 이통사 인터넷 홈페이지를 이용하여 소액결제를 원천적으로 차단하거나 결제금액 제한
    • - 자신의 스마트폰으로 114를 눌러 상담원과 연결해도 소액결제 차단 가능
  4. 4. 스마트폰용 백신프로그램을 설치하고 주기적으로 업데이트
  5. 5. T스토어 · 올레마켓 · LGU+ 앱스토어 등 공인된 오픈마켓을 통해 앱 설치
  6. 6. 보안강화 · 업데이트 명목으로 금융정보를 요구하는 경우 절대 입력 금지

피해구제

    • 지인에게서 온 문자메시지라도 인터넷주소가 포함된 경우 클릭 전에 전화 확인
  • 악성파일 삭제

피싱(Phishing) 이란?

개인정보(Private data)와 낚시(Fishing)의 합성어

피싱 방법 (이미지 위치)

피해유형

“9월 카드 거래내역”이라는 제목의 이메일에서 안내하는 인터넷주소를 클릭하자 가짜 은행사이트로 접속되면서 보안카드번호 전부를 입력한 결과,
범행계좌로 무단 이체

피해 예방 방법 (@디자인필요)

  1. 1. OTP(일회성 비밀번호생성기), 보안토큰(비밀정보 복사방지) 사용
  2. 2. ‘출처불명’ 또는 금융기관 주소와 ‘다른 주소’로 발송된 이메일 즉시 삭제

    이메일 첨부파일에 확장자가 ‘.exe, .bat, .scr 등 압축파일’이면 열람 금지

  3. 3. 보안카드번호 전부를 절대 입력하지 말 것
  4. 4. 사이트 주소의 정상 여부 확인
  5. 5. 보안강화 · 업데이트 명목으로 금융정보를 요구하는 경우 절대 입력 금지

    가짜 사이트는 정상 사이트 주소와 유사하나, 문자열 순서·특수문자 삽입 등에서 차이 있음

대처요령

금전피해가 발생한 경우
  • 피해구제
    • 신속히 112센터나 금융기관 콜센터를 통해 지급정지 요청 후 해당 은행에 경찰이 발급한 ‘사건사고 사실확인원’을 제출하여 피해금 환급 신청
  • 피해금 환급 과정(전기통신금융사기 피해금 환급에 관한 특별법)
    피해금 환급 과정 (이미지 위치)

    가짜사이트 신고 : 한국인터넷진흥원 보호나라(www.boho.or.kr)에 신고

금전 피해가 발생하지 않은 경우
  • 수신한 이메일 삭제
  • 입력했던 금융정보들은 해당 금융기관을 통해 변경
  • OTP(일회성 비밀번호생성기) 사용

피싱의 진화된 형태 파밍사이트 식별법과 대처방법에 대해 알려드립니다.

파밍(Pharming) 이란?

악성코드에 감염된 사용자PC를 조작하여 금융정보를 빼내는 것

파밍 방법 (이미지 위치)

(피싱사이트) ‘http://*Kb*bank.com’ 등 정상 홈페이지로 가장하여 금융정보(보안카드번호 전부) 입력을 요구, 신종금융사기의 주요 범행수단

피해유형

피해자 184명이 동일한 파밍 수법으로 금융정보가 탈취되고 공인인증서가 재발급되어 불시에 13억원이 무단 이체됨

피해 예방 방법 (@디자인필요)

  1. 1. OTP(일회성 비밀번호생성기), 보안토큰(비밀정보 복사방지) 사용
  2. 2. 컴퓨터·이메일 등에 공인인증서, 보안카드 사진, 비밀번호 저장 금지
  3. 3. 보안카드번호 전부를 절대 입력하지 말 것
  4. 4. 사이트 주소의 정상 여부 확인

    가짜 사이트는 정상 사이트 주소와 유사하나, 문자열 순서 · 특수문자 삽입 등에서 차이 있음

  5. 5. 윈도우, 백신프로그램을 최신 상태로 업데이트하고 실시간 감시상태 유지
  6. 6. 전자금융사기 예방서비스(공인인증서 PC지정 등) 적극 가입(9. 26 전면 시행 예정)
  7. 7. ‘출처불명’ 한 파일이나 이메일은 즉시 삭제 및 무료 다운로드 사이트 이용 자제

스팸메시지 대응하기

휴대폰, 전자우편, 메신저 등 다양한 매체를 통해 수신되는
불법스팸 메시지들에 대한 대처법을 알려 드립니다.

스팸 메시지(Spam Message)란?

수신자의 동의 없거나 수신동의 의사를 철회하였음에도 불구하고 불특정다수에 대량으로 발송되는 문자메시지 도는 음성전화로써, 광고/대출 등 무차별적인 정보 제공을 목적으로 행해지는 행위를 뜻 함

스팸 메시지 차단법

스팸등록을 통한 스팸 메시지 차단
  • 스마트폰이 가진 기본 기능 사용 (안드로이드 기기만 해당)
    스팸 메시지를 클릭하면 메시지 옵션 버튼 창이 나타나며,
    버튼 창에서 스팸 등록 버튼을 누르게 되면 문자를 보낸 번호가 스팸으로 등록되어 차단됨
  • 하지만 다른 번호로 온 스팸 문자는 차단되지 않는 단점이 있으며, 스팸 메시지 번호를 하나하나 등록해야 하는 번거로움 있음
어플리케이션을 통한 스팸 메시지 차단
스마트폰에 설치할 수 있는 어플리케이션을 사용
  • ‘후후- 대한민국 스팸 잡는 1등 전화’ (안드로이드 기기 대상), ‘후후- 모르는 전화번호, 스팸번호 확인’ (아이폰 기기 대상) 어플리케이션 설치
  • 어플리케이션 특징
    • 발신자 전화번호 정보 즉시 확인
      • 대한민국 2천만 개 이상의 전화번호 정보 즉시 확인
      • 상호, 업종, 지역 등 전화번호의 상세정보 확인
      • 공중전화, 발신전용 전화 등 전화번호 종류 확인
    • 스팸전화 등 악성 전화번호 정보 확인
      • 스팸 전화, 스팸 메시지, 보이스피싱, 스미싱 번호 등 확인
