Script Option

option base value note
current 1 처음에 열려있을 컨텐츠 선택. 1,2,3.... 식으로 숫자로 선택. 0 이면 모두 닫혀있음. 'all'이면 모두 열린상태
toggle true true or false 선택. true일 경우 하나가 열리면 다른 아코디언탭은 자동으로 닫힘.
scrollTop true true or false 선택. true일 경우 열리면 상단으로 자동스크롤
addTop 0 상단 스크롤 시 추가적으로 줄 여백
