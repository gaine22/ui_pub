label Elements

attributes base value note
class ui_radio, ui_checkbox 기본상태, 기본 유지되는 class
data-all-check="group name" 전체체크 사용시 해당 input check box 에 name 속성 추가하여 data-all-check 와 동일한 name 설정
checked checked="checked" 상태
disabled disabled="disabled" 상태
foucs focus 상태
ui-name input의 name 값 radio일때 생성
ui-id input의 id 값 radio, checkbox 생성

Elements

attributes base value note
data-select-id unique id 유니크한 id값으로 clone select 생성 및 동기화
title select title clone select의 제목값으로 설정

Option

option base value note
id data-select-id 값 동적인 option 추가 및 수정에 인한 리셋이 필요한 셀렉트
