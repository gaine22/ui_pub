Script Option

option base value note
view 1 화면에 노출 아이템 갯수. 기본 1
current 1 처음 노출되는 아이템. 1,2,3..... 으로 선택. 멀티일 경우(view : 2이상일 경우) 현재 미적용
type 'slide' 'slide' or 'fade' or 'cover', 아이템 바뀌는 효과 선택. *멀티 뷰일 경우 slide만 적용됨
item_eff false true or false, item 내부 css 효과
easing 'easeOutQuart' * linear,
* easeInQuad, easeOutQuad, easeInOutQuad, easeOutInQuad,
* easeInCubic, easeOutCubic, easeInOutCubic, easeOutInCubic,
* easeInQuart, easeOutQuart, easeInOutQuart, easeOutInQuart,
* easeInQuint, easeOutQuint, easeInOutQuint, easeOutInQuint,
* easeInSine, easeOutSine, easeInOutSine, easeOutInSine,
* easeInExpo, easeOutExpo, easeInOutExpo, easeOutInExpo,
* easeInCirc, easeOutCirc, easeInOutCirc, easeOutInCirc,
* easeInElastic, easeOutElastic, easeInOutElastic, easeOutInElastic,
* easeInBack, easeOutBack, easeInOutBack, easeOutInBack,
* easeInBounce, easeOutBounce, easeInOutBounce, easeOutInBounce
easing_re 'easeInOutQuart' 반대움직임 효과가 달라야할때. cover만 사용
speed 500 아이템이 변경되는 시간간격
rolling true true or false. 무한반복 여부 선택
dot true true or false, 페이징 사용 여부
nav true true or false, 이전다음 버튼 사용 여부
responsive false true or false. true일때만 view_... 적용.
└ view_mobile [0, 1] [device가로 사이즈, 보여줄 갯수]
└ view_tablet [760, 2] [device가로 사이즈, 보여줄 갯수]
└ view_desktop [1024, 3] [device가로 사이즈, 보여줄 갯수]
└ view_wide [1400, 4] [device가로 사이즈, 보여줄 갯수]
autoplay true true or false, 자동진행 여부 선택
└ autoplay_state 'play' autoplay ture 선택 시 'play' or 'stop' 중 택1.
└ autoplay_time 5000 자동진행 시간간격
mouseDrag true true or false, pc에서 마우스 드래그 사용여부
touchDrag true true or false, 모바일 터치 사용여부
└ speed_drag 200 터치 및 드래그 완료,취소 이벤트 후 나머지 남은 거리 움직임 속도
load false true or false, 외부파일 로드
└ load_url null 외부파일 주소
└ load_type null 파일 타입, 'json', 'html' 등
└ load_view 1 불러올 item 갯수, 'all' 전체 불러옴
└ load_start 0 처음 보여줄 시작 아이템
