ul.list-bullet

  • 스마트폰이 가진 기본 기능 사용 (안드로이드 기기만 해당)
    스팸 메시지를 클릭하면 메시지 옵션 버튼 창이 나타나며,
    버튼 창에서 스팸 등록 버튼을 누르게 되면 문자를 보낸 번호가 스팸으로 등록되어 차단됨
  • 하지만 다른 번호로 온 스팸 문자는 차단되지 않는 단점이 있으며, 스팸 메시지 번호를 하나하나 등록해야 하는 번거로움 있음
